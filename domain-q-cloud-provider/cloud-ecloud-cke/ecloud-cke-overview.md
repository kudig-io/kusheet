# 移动云 CKE (China Mobile Cloud Kubernetes Engine) 概述

## 产品简介

移动云Kubernetes引擎是中国移动云提供的企业级容器服务，依托中国移动强大的网络基础设施，为企业客户提供高性能、高可靠的容器化应用管理平台。

## 核心特性

### 运营商级保障
- 中国移动网络基础设施支撑
- 电信级SLA服务保障
- 全国范围网络覆盖优势
- 专业运维团队支持

### 高性能架构
- 支持大规模集群管理
- 智能调度算法优化
- 网络延迟深度优化
- 计算资源弹性伸缩

## 架构组件

### 控制平面
- **高可用Master集群**: 多可用区部署
- **分布式etcd存储**: 高性能状态管理
- **智能调度引擎**: 基于负载的调度优化
- **自动化控制器**: 运维自动化组件

### 节点类型
- **弹性计算节点**: 灵活的计算资源配置
- **GPU加速节点**: AI/ML计算场景支持
- **边缘计算节点**: CDN边缘计算优化
- **专属宿主机**: 物理资源隔离

## 网络架构

### 网络方案
- **VPC网络集成**: 与移动云VPC深度融合
- **高性能网络插件**: 自研CNI网络优化
- **网络策略管理**: Kubernetes网络策略支持
- **SD-WAN集成**: 软件定义广域网

### 负载均衡
- **SLB负载均衡**: 移动云弹性负载均衡
- **内网负载均衡**: 内部服务高效访问
- **公网负载均衡**: 外部服务安全暴露
